## Instalación

Asegúrate de tener Ruby y Bundler instalados en tu sistema. Luego, sigue estos pasos:

```bash

git clone https://github.com/tu_usuario/tu_proyecto.git

# Entra al directorio del proyecto
cd tu_proyecto

# Instala las dependencias
bundle install
